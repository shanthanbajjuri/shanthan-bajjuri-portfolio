Projects

Wi-Fi Home Automation using Blynk IoT (2023)

This home automation project demonstrated a fully functional IoT system for remotely controlling household appliances using the Blynk platform and a Wi-Fi-enabled microcontroller (ESP8266/ESP32). The microcontroller was programmed via Arduino IDE using the Blynk library and a unique Authentication Token to securely connect to the Blynk Cloud Server. Users interacted through the Blynk mobile app, which sent commands to virtual pins linked to GPIO outputs. These outputs activated relay modules to control high-voltage appliances like fans, lights, and doors.

Technologies: ESP8266/ESP32, Arduino IDE, Blynk App, Relay Module, GPIO

Role: Project designer and embedded programmer

IoT-Based ATM Security System (2024)

Designed a multi-sensor IoT security system for ATMs to detect physical tampering and environmental threats. The system used vibration, tilt, and temperature sensors connected to an ESP32 microcontroller. Upon detecting anomalies, it triggered alarms, locked the ATM, and sent real-time alerts—including GPS location and threat type—via SMS and cloud protocols to bank security and law enforcement.

Technologies: ESP32, Vibration Sensor, Tilt Sensor, Temperature Sensor, GSM/Wi-Fi, GPS

Role: System architect and IoT developer

Employee Attrition Analysis using People Analytics (2025)

A Master's level project focused on analyzing employee turnover using survey data and HR metrics. Conducted Exploratory Data Analysis (EDA) and predictive modeling in R to identify key factors behind voluntary attrition. Logistic Regression and Random Forest models revealed that lack of career growth and salary dissatisfaction were top predictors. The insights helped segment employees by attrition risk and propose targeted retention strategies.

Technologies: R, ggplot2, dplyr, Logistic Regression, Random Forest

Role: Data analyst and model developer
